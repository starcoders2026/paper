# Dataset Notes

This folder contains notes about the dataset structure and data preparation process used in the image classification project.

## Dataset Purpose

The dataset was used to train and evaluate an image classification model for category-based image analysis.

## Data Preparation Steps

- Collected image data for the target classes
- Organized images into class-based folders
- Checked image quality before training
- Removed problematic or unclear images when needed
- Prepared the dataset for model training and evaluation

## Notes

The actual image dataset may not be fully included in this repository because of file size, privacy, or source limitations.
