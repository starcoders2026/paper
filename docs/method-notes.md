# Method Notes

## Overview

This file explains the general method used in the image classification project.

## Main Steps

1. Prepare the image data
2. Organize the classes
3. Apply preprocessing steps
4. Train the model
5. Evaluate the model performance
6. Review incorrect predictions
7. Document the results

## Model Development Process

The model development process follows a step-by-step workflow. First, the data is prepared and checked. Then, the model is trained and evaluated. After that, incorrect predictions are reviewed to better understand the weak points of the model.

## Evaluation Focus

The evaluation process focuses on checking how well the model performs on validation and test data.
