# README Review Checklist

## Purpose

This checklist is used to review whether the main README file explains the project clearly.

## Checklist

- [ ] Project title is clear
- [ ] Project purpose is explained
- [ ] Repository structure is shown
- [ ] Main folders are explained
- [ ] Tools and methods are listed
- [ ] Current project status is included
- [ ] Setup or review steps are mentioned
- [ ] Future updates are noted
